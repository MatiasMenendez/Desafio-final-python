App de blog alumno Matias Menendez.

La app se inicializa con la url AppBlog/inicio a partir de esa view se puede acceder a todas las demas

AppBlog/about  url about
AppBlog/pages url Blogs
AppBlog/inicio url index

CRUD
AppBlog/blogFormulario url para agregar nuevo blog y subirloa la base
AppBlog/EliminarBlog url para eliminar blog de la base
AppBlog/pages url para listar blogs

todas las url heredan padre.html

users/register url registra nuevos usuarios
